
# Customer Churn Analysis

## Objective

Analyze customer data to identify patterns behind subscription cancellations (churn).

## Dataset

Telco Customer Churn Dataset

## Analysis Performed

* Data loading and inspection
* Missing value handling
* Churn distribution analysis
* Contract type analysis
* Tenure analysis
* Monthly charges analysis
* Internet service analysis
* Payment method analysis

## Key Findings

* Month-to-month customers have higher churn.
* Newer customers are more likely to leave.
* Higher monthly charges are associated with increased churn.
* Contract type significantly impacts retention.

## Recommendations

* Encourage long-term contracts.
* Improve onboarding for new customers.
* Offer loyalty programs and retention incentives.
* Enhance customer support and engagement.
